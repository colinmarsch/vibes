# MileStones

`milestones/` is the scaffold for a static website that will host running training plans in multiple formats over time.

## Current contents
- `index.html` — starter landing page
- `styles.css` — base styling for the site

## Next ideas
- Add route-like pages by folder (e.g., `plans/5k/index.html`)
- Add downloadable plan formats (PDF/CSV/Markdown)
- Add a simple index of available plans
