# Doji Vu

**Doji Vu** is a tiny local-first web app for tracking repeatable life tasks such as:

- 🪴 Watering plants
- 🗑️ Taking out garbage
- ☎️ Calling parents
- ✅ ...or anything else with a name + emoji

## Features

- Add unlimited custom trackers (name + emoji)
- Mark any tracker as done in one click
- See total completion count, the last completion timestamp, and a simple per-day average
- Delete individual trackers or clear all data
- Persists in the browser using `localStorage`

## Run locally

Open `index.html` in a browser.
